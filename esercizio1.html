Ripasso Formattazione Logica

La formattazione logica in HTML è utilizzata per dare significato al testo. Ogni elemento ha uno scopo specifico e aiuta a strutturare semanticamente il contenuto. Ecco alcuni esempi comuni:

  • Strong: per enfatizzare fortemente il testo. Esempio: Attenzione!
  • Emphasis: per enfatizzare leggermente il testo. Esempio: Questo è davvero importante.
  • HTML: per abbreviazioni. Passa il mouse sopra per vedere il significato completo.
  • Citation: per citazioni di titoli di opere. Esempio: La Divina Commedia di Dante Alighieri.
  • Code: per blocchi di codice. Esempio: console.log("Hello World!");
  • Definizione: per definizioni di termini. Esempio: HTML è il linguaggio di markup per la creazione di pagine web.
  • Input da tastiera: per input da tastiera. Esempio: Premi Ctrl + C per copiare.
  • Highlight: per evidenziare il testo. Esempio: Questa è una parte importante del testo.
  • Output di un programma: per output di programmi. Esempio: Error 404: Page not found
  • Variabile: per variabili matematiche o di programmazione. Esempio: L'equazione è y = mx + b
